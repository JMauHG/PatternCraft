# Creational Patterns

Creational patterns deal with object creation mechanisms, trying to create objects in a manner suitable to the situation.

## The Patterns

| Pattern | Mnemonic |
|---------|----------|
| **Singleton** | A country can have only one official government |
| **Builder** | Building a custom burger with optional toppings |
| **Factory Method** | A logistics company that can ship by truck or ship |
| **Abstract Factory** | A furniture factory that makes matching chairs and sofas |
| **Prototype** | Copying a document instead of creating from scratch |
