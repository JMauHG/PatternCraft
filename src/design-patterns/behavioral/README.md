# Behavioral Patterns

Behavioral patterns are concerned with algorithms and the assignment of responsibilities between objects. They describe not just patterns of objects or classes but also the patterns of communication between them.

## The Patterns

| Pattern | Mnemonic |
|---------|----------|
| **Chain of Responsibility** | A tech support call that gets escalated from level 1 to level 2 to level 3 |
| **Command** | A restaurant order ticket that encapsulates a request to the kitchen |
| **Iterator** | A TV remote's "next channel" button that hides how channels are stored |
| **Mediator** | An air traffic controller coordinating planes that never talk directly |
| **Memento** | A video game save point you can roll back to after dying |
| **Observer** | A newspaper subscription — publisher notifies all subscribers of new issues |
| **State** | A traffic light that behaves differently in red, yellow, and green states |
| **Strategy** | A GPS app offering driving, walking, or cycling route strategies |
| **Template Method** | A recipe with fixed steps where some ingredients can be customized |
| **Visitor** | A tax auditor visiting different shops, applying rules specific to each |
